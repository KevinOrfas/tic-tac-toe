# Multiplayer Tic-Tac-Toe Server

Backend server for the multiplayer Tic-Tac-Toe game using Socket.IO for real-time communication and PostgreSQL for game persistence.

## Available Scripts

- `npm run dev` - Start development server with hot reload (using tsx watch)
- `npm run build` - Build TypeScript to JavaScript for production
- `npm start` - Start production server (requires build first)
- `npm test` - Run tests in watch mode
- `npm run test:coverage` - Run tests with coverage report

## Tech Stack

- **Socket.IO** - Real-time bidirectional communication
- **PostgreSQL** - Database for storing game history
- **TypeScript** - Type-safe development
- **tsx** - TypeScript execution and watch mode
- **Vitest** - Testing framework
